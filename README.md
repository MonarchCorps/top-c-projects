# Top C Projects

A collection of low-level C projects I built while developing EduOS, the Linux OS version of Edgai. Everything here is written from scratch, no frameworks, no shortcuts.

| Project | Description | Link |
|---|---|---|
| Mini Shell | A Unix shell that parses and executes commands | [repo](https://github.com/MonarchCorps/mini-shell) |
| Memory Allocator | Custom malloc/free implementation | [repo](https://github.com/MonarchCorps/memory-allocator) |
| Process Monitor | Tracks and displays running system processes | [repo](https://github.com/MonarchCorps/process-monitor) |
| Terminal Text Editor | A text editor that runs entirely in the terminal | [repo](https://github.com/MonarchCorps/terminal-based-text-editor) |
| Terminal Chat App | TCP-based chat application for the terminal | [repo](https://github.com/MonarchCorps/chat-application) |
| File Explorer | Navigate and manage files from the terminal | [repo](https://github.com/MonarchCorps/file-explorer) |
| Package Manager | Installs and manages software packages | [repo](https://github.com/MonarchCorps/package-manager) |
